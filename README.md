# SQL_Tool
Spring boot이용해서 SQL Tool 만들어보기
- h2 database 연동하여 CRUD 기능 구현
- PostgreSQL, mariaDB 둘 중 하나의 데이터베이스로 접속, 접속 종료, 쿼리 수행 기능 구현
- 접속 로그 기록 기능 구현
- 쿼리 로그 기록 기능 구현
- view는 postman 이용
- JPA, JWT 활용
